## ❓ FAQ

### Q: 신형 사이트를 지금 바로 사용해도 되나요?
**A**: 아니요. 개발 모드 노출로 인한 심각한 보안 위험이 있습니다. Phase 1 조치를 완료한 후 사용하세요.

### Q: 접근성 개선에 얼마나 걸리나요?
**A**: 기본 수준(WCAG Level A): 2-4주, 권장 수준(WCAG Level AA): 2-3개월

### Q: 비용은 얼마나 드나요?
**A**: 프로젝트 규모에 따라 다르지만, 전문 접근성 컨설팅 포함 시 1,000만원~3,000만원 수준

### Q: 어떤 사이트를 기반으로 개선해야 하나요?
**A**: 신형 사이트의 프레임워크를 사용하되, 구형 사이트의 접근성 기능을 모두 이식하는 것을 권장합니다.
